GitGator

GitGator is a dashboard for CTO, Director of Development, Development Manager and Development Leads. If you are senior leader managing a large development team, who is involved technically and would like to know what is happening day to day? would like to keep hands on the pulse of team than this tool is for you.

This is a tool to give you a high level heatmap of development team. Remove all the clutter of git where you have many tools(Bots) creating and closing PR.

Use GitGator one screen, a quick glance to know:

  • How many pull request created today, last seven days and last 30 days.
  • How many pull request closed today, last seven days and last 30 days.
  • Monitor multiple organizations Pull Request/Jira Tickets from one screen.
  • Monitor multiple developers from one screen.
  • Drill down in a Jira tickets for the developer.
  • Zero down to one repository to see all the pull request.
  • One click to drill down view of Jira and Git.
  • Watch today's pull request details scrolling on top of the screen

This is not a replacement of Jira/Git, but a tool which bring all the needed details on one screen for a senior executive.
